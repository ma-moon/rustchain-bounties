# Elyan Labs Ecosystem Resources

This document provides an overview of key projects and resources from [Elyan Labs](https://github.com/Scottcjn), the organization behind RustChain and related tools.

## Core Projects

### RustChain
- **Website**: https://rustchain.org
- **GitHub**: https://github.com/Scottcjn/RustChain
- **Description**: A Proof-of-Antiquity blockchain where vintage hardware earns higher mining rewards. RustChain uses hardware attestation to create a unique consensus mechanism that rewards older, more "antique" hardware.
- **Relevance**: This is the foundational blockchain protocol for the entire ecosystem. All bounties in this repository are related to contributing to RustChain development and community growth.

### BoTTube
- **Website**: https://bottube.ai
- **Description**: An AI video platform where 119+ agents create content autonomously. BoTTube demonstrates practical applications of AI agents in content creation and video production.
- **Relevance**: Shows how AI agents can be integrated with blockchain technology for decentralized content creation.

### Beacon Atlas / Beacon
- **Website**: https://rustchain.org/beacon/
- **Description**: Beacon provides network monitoring and visualization tools for the RustChain network.
- **Relevance**: Essential for node operators and developers who want to monitor network health and performance.

### Grazer
- **GitHub**: https://github.com/Scottcjn/grazer-skill
- **Description**: A skill/agent framework for autonomous operations.
- **Relevance**: Useful for developers building automated tools and agents that interact with the RustChain ecosystem.
